# Options:

```
      --help     Muestra ayuda                                        [booleano]
  -b, --base     Es la base la tabla de multiplicar         [número] [requerido]
  -l, --listar   Muestra por consola la tabla        [booleano] [defecto: false]
  -h, --hasta    Es hasta donde se multiplica la base     [número] [defecto: 10]
  -v, --version  Muestra número de versión                            [booleano]
```
